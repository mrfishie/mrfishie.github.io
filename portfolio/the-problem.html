THE PROBLEM

These images are all for parts of the site that were designed and developed by me.

The homepage for users who aren't yet logged in.

The homepage for logged-in users.

The comment system, visible on a bug page. Administrators can modify and delete all comments, and can also change permissions on comments and bugs.

The 'Overview' tab of the site admin page. The site name and visibility can both be changed here.

The 'Sections' tab of the site admin page. Sections can be added or modified here, and sections that may be 'in trouble' (e.g. high percentage of open bugs) are highlighted in red.

A section open for modification. Due to time limitations, only the list of developers can be changed.

Creating a new section. The name and description can be set, and developers can be added.

The 'Users' tab of the site admin page. User ranks can be changed here (ranks are Unverified, Standard and Administrator - these affect permissions), and other info can be previewed.

The 'Permissions' tab of the site admin page. Global ranks can be set here, and ranks can also be set on a per-section, bug, or comment basis.

Modifying the permissions for an individual bugs. Permissions are set on a rank basis, with an additional whitelist.
